Notes

we can use the below classes for validation based on state change

  1. ng-touched: control visit state
  2. ng-untouched: control unvisited state
  3. ng-dirty: value has been changed
  4. ng-pristine: value has not been changed
  5. ng-valid: values are valid
  6. ng-invalid: values are invalid
